# **App Name**: CommerceManager Pro

## Core Features:

- Creación de Ventas: Crear transacciones de venta, gestionar múltiples productos por venta y calcular automáticamente subtotales, impuestos y totales.
- Generación de Facturas: Generar recibos imprimibles y mantener un registro histórico de todas las transacciones de venta.
- Gestión de Stock: Añadir, eliminar y modificar productos con actualizaciones automatizadas de control de stock en cada venta y alertas de stock bajo.
- Seguimiento de Inventario: Registrar todos los movimientos de inventario para mantener niveles de stock precisos.
- Gestión de Clientes: Registrar clientes, rastrear su historial de compras, gestionar deudas y procesar pagos parciales.
- Estados de Cuenta: Permitir a los usuarios ver los saldos pendientes y generar estados de cuenta imprimibles para cada cliente.
- Análisis del Panel: Mostrar métricas clave como las ventas diarias, los productos con bajo stock y los clientes con deudas pendientes.

## Style Guidelines:

- Color primario: Verde suave (#A7D1AB) para evocar confianza y eficiencia.
- Color de fondo: Verde muy claro (#F5F9F6) para una estética limpia y profesional.
- Color de acento: Verde amarillento apagado (#C4D7A8) para resaltados y elementos interactivos, proporcionando un contraste sutil.
- Fuente del cuerpo y del encabezado: 'Inter' sans-serif, conocida por su legibilidad y estética limpia, ideal tanto para encabezados como para texto del cuerpo en sistemas administrativos.
- Diseño responsivo con una barra lateral para la navegación en el escritorio y un menú de hamburguesa en los dispositivos móviles para garantizar una visualización óptima en varios tamaños de pantalla.
- Utilizar iconos simples y claros para la navegación y las acciones clave dentro de los módulos para mejorar la usabilidad.
